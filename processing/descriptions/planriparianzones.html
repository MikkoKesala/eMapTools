Plan riparian buffer zones. This version works only in Finland as it uses pre-modelled data sets for retention of suspended solids. The method accumulates RUSLE values by the massflux method and determines cost values for the retention of suspended solids from a water body.

Input parameters

  • Planning area Planning or cutting area as vector polygon format
  • Retention of suspended solidThe minimum percentage of suspended solids that should be retained in a riparian zone
  • Minimum widthMinimum width of riparian zone
  • Average widthAverage width of riparian zone

Outputs

  • Riparian zoneProposed riparian zone based on accumulation of suspended solid
