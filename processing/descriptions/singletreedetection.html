Detects individual trees from the canopy model and enriches the canopy model data (e.g. canopy crown) to the output level. The result is trees as vector points. Individual tree tops using Minimum Curvature-Based Tree Detection (Kaartinen et al. 2012). The algorithm calculates NDVI statistics from the orthoimage.

Input parameters

  • CHM Canopy height model (m)
  • CIR Color infrared aerial image (band1 = infrared,band2 = red, band3 = green)

Outputs

  • Trees Single-trees as vector points with attibute data from CHM and CIR
